FlashTalk

Conecte-se rapidamente com colegas através de salas temporárias de chat.

Ideia Principal

O FlashTalk é um mini chat temporário pensado para facilitar conversas instantâneas entre estudantes durante trabalhos em grupo ou monitorias. Cada sala expira após um tempo definido, garantindo foco e privacidade.

Funcionalidade Principal

Gerar uma sala efêmera com um código de acesso simples, permitindo que usuários troquem mensagens em tempo real sem cadastro.

  • Crie uma sala temporária em segundos.
  • Compartilhe o código para convidar participantes.
  • Envie mensagens instantâneas enquanto a sala estiver ativa.

Benefícios

Agilidade

Ideal para tirar dúvidas rápidas ou organizar tarefas pontuais.

Privacidade

As salas expiram automaticamente, evitando histórico permanente.

Usabilidade

Interface direta com foco no que importa: conversar.
