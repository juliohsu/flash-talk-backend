# Backend FlashTalk

Repositório responsável pelo backend do FlashTalk, um chat temporário voltado a interações rápidas entre estudantes. A aplicação utiliza Node.js com Express para expor a lógica de negócio e entregar um demo com a página estática localizada em `public/`.

O servidor ficará acessível em `http://localhost:3000`. A rota `/hello` responde com o texto **Hello world!**.# flash-talk-backend
